## Tic - Tac - Toe
A simple Tic-Tac-Toe game built with plain HTML, CSS and JavaScript


## 🚀 Features

- ✅ Win/draw detection
- ✅ 2-player local game (Player X and Player O)
- ✅ Restart button
- ✅Simple,clean and responsive UI


## 🛠️ Tech Stack
- **HTML5** – Structure  
- **Tailwind CSS** – Styling  
- **JavaScript** – Logic and interactivity
